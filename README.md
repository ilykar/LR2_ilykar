# Лабораторная работа №2 - Управление умным домом 
 
## Описание проекта 
Консольное приложение для управления устройствами умного дома 
 
## Функции: 
- Добавление/удаление устройств 
- Просмотр коллекции 
- Сохранение в XML 
- Поиск и фильтрация 
 
## Основные команды: 
- help - справка 
- insert - добавить устройство 
- show - показать все устройства 
- save - сохранить данные 
- exit - выход
